## Unit Converter

### Convert length weight or temperature units.

## Supported Units
- Length: mm, cm, dm, m, km, in, ft, yd, mi

- Weight: mg, g, kg, oz, lb

- Temperature: C, F, K

## Usage

### Install packages

```bash
python -m pip install -r requirements.txt
```

### Run it locally
```bash
python unit-converter.py
```
> then, visit in the browser the url: http://127.0.0.1:8080
